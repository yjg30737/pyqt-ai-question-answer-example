# pyqt-ai-question-answer-example
Example of AI question and answer model (deepset based, such as roberta-base-squad2) with Python desktop application

## Requirements
* PyQt5>=5.14
* huggingface_hub
* transformers

## How to Run
1. git clone ~
2. pip install -r requirements.txt
3. python main.py
